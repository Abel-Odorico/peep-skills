## Veredito

| Nota | Status |
|---:|---|
| 95-100 | Mestre absoluto |
| 90-94 | Excelente |
| 85-89 | Aprovado com ajustes finos |
| 70-84 | Reprovado temporariamente |
| 50-69 | Fraco |
| 0-49 | Crítico |

## Penalidades automáticas

- `noindex` acidental: -20
- robots bloqueando site inteiro: -25
- sitemap ausente em site médio/grande: -5
- Home sem proposta clara: -10
- página comercial sem CTA: -8
- conteúdo copiado: -20
- schema falso: -20
- LCP > 4s: -10
- CLS > 0,25: -8
- ausência de contato em site comercial: -10
- ausência de política de privacidade em site com coleta de dados: -5

## Regra de aprovação

Nota >= 85 e nenhum bloqueador crítico ativo.
