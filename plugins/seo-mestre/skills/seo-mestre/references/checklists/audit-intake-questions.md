# Perguntas de Intake — SEO MESTRE

## Sobre o negócio

1. Qual é o nome da marca?
2. O que ela vende ou oferece?
3. Quem é o público-alvo?
4. Qual região atende?
5. Qual é o objetivo principal do site?
6. Qual conversão importa mais: WhatsApp, formulário, ligação, compra, cadastro ou orçamento?

## Sobre o site

1. Qual é a URL?
2. É WordPress, HTML, Laravel, Next.js, Astro, Shopify, WooCommerce ou outro?
3. O site já está publicado?
4. Existe Google Search Console?
5. Existe Bing Webmaster Tools?
6. Existe Analytics?
7. Existe Google Business Profile?

## Sobre SEO

1. Quais palavras ou temas são importantes?
2. Quais concorrentes incomodam?
3. O site já recebe tráfego orgânico?
4. Quais páginas são mais importantes?
5. Existe blog ou central de conteúdo?
6. Há cases, depoimentos ou provas?

## Sobre IA

1. Quer aparecer em respostas de IA?
2. Quer permitir crawlers de IA?
3. Quer bloquear bots de treinamento?
4. Quer criar llms.txt?
5. Quer criar páginas Markdown resumidas?

## Sobre performance

1. O site usa muitos scripts?
2. Usa Elementor ou construtor visual?
3. Usa vídeos pesados?
4. Usa chat, pixels, mapas, embeds?
5. Tem CDN/cache?
