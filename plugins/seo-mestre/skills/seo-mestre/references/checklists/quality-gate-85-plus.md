# Quality Gate 85+ — SEO MESTRE

A entrega só é considerada aprovada se todos os critérios abaixo forem verdadeiros.

## Bloqueadores zerados

- [ ] Não existe `noindex` acidental
- [ ] Robots não bloqueia páginas importantes
- [ ] Sitemap existe e é coerente
- [ ] Páginas comerciais têm H1
- [ ] Páginas comerciais têm CTA
- [ ] Home explica o negócio
- [ ] Contato existe
- [ ] Schema não é falso
- [ ] Conteúdo principal é acessível
- [ ] Performance mínima é aceitável

## Mínimos técnicos

- [ ] Lighthouse SEO = 100 ou justificativa forte
- [ ] Performance >= 90 ou plano técnico claro
- [ ] Accessibility >= 90
- [ ] Best Practices >= 90
- [ ] LCP <= 2,5s
- [ ] INP <= 200ms
- [ ] CLS <= 0,1

## Mínimos de conteúdo

- [ ] Resposta direta nas páginas principais
- [ ] FAQ real
- [ ] Proposta de valor específica
- [ ] Provas reais ou lista de provas necessárias
- [ ] Diferenciais claros
- [ ] Páginas de serviço/produto específicas

## Mínimos de IA

- [ ] Estratégia de crawlers definida
- [ ] Páginas rastreáveis por agentes
- [ ] HTML semântico
- [ ] Links reais
- [ ] llms.txt avaliado
- [ ] Conteúdo com citabilidade

## Veredito

- [ ] Nota >= 85
- [ ] Sem bloqueadores críticos
- [ ] Roadmap para 100/100
